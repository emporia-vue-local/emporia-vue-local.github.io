## Supported Hardware
Clearly state support for Vue 2 and Vue 3.

## Prerequisites (Bill of Materials)

### Hardware list
* USB UART
* Jumper wires
* Headers
* Soldering iron

### Software list
* ESPHome
* esptool
* drivers
